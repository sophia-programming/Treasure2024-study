AWSのサービスを知る

1. IAM（AWS Identity and Access Management ）
→　AWSのリソースへのアクセスを安全に管理するためのサービス
    1. ユーザーとグループの管理：AWSアカウント内でユーザーを作成し、グループに組織化できる
    2. ポリシーの管理：ユーザー、グループに対してアクセス権限を設定する
    3. 一時的な認証情報：一時的な認証情報を発行することで、セキュリティを向上させる
    4. アクセス分析：AWSリソースへのアクセス履歴を監査することができる

2. EC2（Elastic Compute Cloud）
→　クラウド上の仮想サーバーを提供するサービス。サーバーの起動、停止、再起動、削除が可能
    1. インスタンスタイプ：インスタンスの性能を選択できる
    2. インスタンスステータス：インスタンスの状態を確認できる
    3. インスタンスストア：インスタンスに一時的にデータを保存できる
    4. インスタンスの起動：AMI（Amazon Machine Image）を指定してインスタンスを起動できる

3. S3（Simple Storage Service）
→　オブジェクトストレージサービス。大容量のデータを保存し、アクセス可能。
    1. バケット：ファイルを保存するためのコンテナ
    2. オブジェクト：ファイルやデータを保存する単位
    3. バージョニング：ファイルのバージョン管理を有効にできる
    4. 暗号化：ファイルの暗号化を有効にできる

4. RDS（Relational Database Service）
→　リレーショナルデータベースサービス。MySQL、PostgreSQL、Oracle、SQLServerなど、様々なデータベースエンジンをサポート
